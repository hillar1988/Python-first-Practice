## Python功能：参考对象（Python features: Refer for object）

* 例如，

y = x


y[0] = 5


print(x)



x = [1, 2, 3, 4]

print (x)

输出将会是[5,2,3,4],[1,2,3,4]

--------------------------------------------------
* In Python if you want pass the value from one object to another, = will pass the value by address.
* For example,


  - x = [1, 2. 3, 4]
